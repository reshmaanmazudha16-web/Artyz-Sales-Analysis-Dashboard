## Objective

The main objectives of this project are:

- Monitor overall sales performance and revenue
- Track units sold and total orders
- Identify top-performing cities and products
- Analyze category-wise and occasion-wise sales
- Understand customer purchasing behavior (Retail vs Bulk)
- Provide dynamic filtering for interactive analysis
  
## Tools & Technologies Used

- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - Data Modeling
  - Conditional Formatting
  - Advanced Chart Formatting
  - Excel Formulas (SUMIFS, COUNTIFS, etc.)

### Data Preparation Steps

1. Cleaned raw dataset (removed duplicates, standardized names)
2. Created Pivot Tables for all KPIs
3. Designed visual charts from Pivot Tables
4. Connected slicers to multiple Pivot Tables
5. Applied consistent formatting and dashboard layout design

## Business Insights

- Retail customers contribute 97% of sales
- Crochet and Handmade categories drive major revenue
- Thrissur and Palakkad show strong performance
- Wedding and Birthday occasions generate higher sales
- A few products consistently outperform others

## How to Use

1. Open the Excel dashboard file.
2. Use the slicers on the right panel to filter by:
   - City
   - Occasion
   - Category
3. Observe how all KPIs and charts update dynamically.
4. Analyze trends and product performance.
